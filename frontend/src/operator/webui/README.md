# Operator Web UI

This is the Angular 21 operator web UI used by the Cuttlefish operator
frontend.

Run commands from this directory:

```bash
npm install
npm run start
npm run build
npm run test
npm run lint
```

`npm run start` serves the development UI on `http://localhost:4200/`.
`npm run build` writes production build output under `dist/`.
`npm run lint`, `npm run clean`, and `npm run fix` use the Google TypeScript
Style tooling configured in `package.json`.
